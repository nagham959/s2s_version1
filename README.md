# SignTranslator - React Project

مشروع React لتحويل موقع SignTranslator من HTML إلى React مع Tailwind CSS.

## البنية (Folder Structure)

```
s2s_website/
├── public/
│   └── index.html
├── src/
│   ├── components/        # مكونات React
│   ├── pages/           # صفحات التطبيق
│   ├── App.js           # المكون الرئيسي
│   ├── index.js         # نقطة الدخول
│   └── index.css        # ملفات CSS و Tailwind
├── package.json
├── tailwind.config.js   # إعدادات Tailwind مع الألوان والخطوط
├── postcss.config.js
└── README.md
```

## الألوان المستخدمة

تم استخراج جميع الألوان من ملفات HTML الأصلية:

- **Primary**: `#F2593D`
- **Primary Dark**: `#D14026`
- **Primary Hover**: `#D14026`
- **Primary Light**: `#FF7E66`
- **Background Light**: `#ffffff`
- **Background Alt**: `#f9fafb`
- **Background Dark**: `#121212`
- **Surface Light**: `#ffffff`
- **Surface Dark**: `#1c1c1c`
- **Text Main**: `#0e151b`
- **Text Sub**: `#4e7597`
- **Text Secondary**: `#94a3b8`
- والمزيد...

## الخطوط

- **Cairo**: الخط الأساسي المستخدم في جميع الصفحات
- **Material Symbols Outlined**: للأيقونات
